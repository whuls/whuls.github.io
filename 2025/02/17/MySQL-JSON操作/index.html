JSON字段新增与修改

JSON Object

设置数据（强行设置）：JSON_SET

不管JSON Object中是否存在这个键，都为其强行设置一个值。

示例1：新增一个值

示例2：新增多个值

新增数据（若存在则不新增）：JSON_INSERT

如果要新增的键在JSON Object中存在，则不新增，否则新增数据。

示例1：INSERT无效

示例2：INSERT有效

替换数据（若不存在则不替换）：JSON_REPLACE

如果要替换的键在JSON Object中存在，则进行替换，否则不生效。

示例1：REPLACE有效

示例2：REPLACE无效
