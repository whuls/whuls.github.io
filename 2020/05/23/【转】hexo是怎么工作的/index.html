дҪңиҖ…пјҡиөөеҪӘ

дҪ еҸҜиғҪз”ЁиҝҮhexo(жҲ–иҖ…jekyll)жқҘжҗӯе»әиҮӘе·ұзҡ„еҚҡе®ўзҪ‘з«ҷгҖӮйҖҡеёёжҲ‘д»¬еңЁе®үиЈ…гҖҒй…ҚзҪ®е®ҢжҲҗhexoд№ӢеҗҺпјҢеҖҹеҠ©hexoпјҢдёҖиҲ¬йҖҡиҝҮд»ҘдёӢжӯҘйӘӨпјҢе°ұеҸҜд»Ҙе®ҢжҲҗдёҖзҜҮеҚҡе®ўзҡ„зј–еҶҷеҸҠеҸ‘еёғпјҢзңҹжҳҜж–№дҫҝжһҒдәҶпјҡ

еҸҜжҳҜпјҢд»Һhexo initеҲ°hexo deployдёӯй—ҙеҸ‘з”ҹдәҶд»Җд№Ҳе‘ўпјҹдёәдәҶжҗһжё…жҘҡиҝҷдёҖиҝҮзЁӢгҖҒзҗҶи§Јhexoзҡ„е·ҘдҪңеҺҹзҗҶпјҢжң¬ж–Үе°ҶиҜ•зқҖеӣһзӯ”д»ҘдёӢй—®йўҳ:

  • е‘Ҫд»ӨиЎҢдёӯзҡ„hexoжҳҜд»Җд№Ҳ
  • hexoжҳҜжҖҺд№Ҳе°ҶжҲ‘д»¬еҶҷеҘҪзҡ„markdownиҪ¬жҚўжҲҗhtmlзҡ„
  • hexoжҸ’д»¶жҳҜеҰӮдҪ•е·ҘдҪңзҡ„
  • жң¬ең°зҡ„hexoйЎ№зӣ®е’Ңgit pageжңүд»Җд№Ҳе…ізі»

жң¬ж–ҮдёҚжҳҜ:

  • hexoзҡ„е®үиЈ…гҖҒдҪҝз”Ёж•ҷзЁӢ
  • git pageзҡ„дҪҝз”Ёж•ҷзЁӢ

е‘Ҫд»ӨиЎҢдёӯзҡ„hexoжҳҜд»Җд№Ҳ?

hexoйЎ№зӣ®еңЁgithubдёҠе·Із»Ҹжңүи¶…иҝҮ17kзҡ„starдәҶпјҢдҪҶжҳҜдҪ зҹҘйҒ“еҗ—пјҢж—ҘеёёжҲ‘д»¬еңЁе‘Ҫд»ӨиЎҢвҖқж“ҚдҪңвҖқhexoж—¶жүҖиҫ“е…Ҙзҡ„hexo(дҫӢеҰӮhexo init)е№¶дёҚжҳҜиҝҷдёӘ17kдёӘstarзҡ„йЎ№зӣ®! жҳҜзҡ„пјҢжҲ‘д»¬еңЁе‘Ҫд»ӨиЎҢдёӯжүҖиҫ“е…Ҙзҡ„вҖқhexoвҖқе®һйҷ…жҳҜhexo-cliйЎ№зӣ®пјҢиҜҘйЎ№зӣ®еңЁgithubдёҠзҡ„starиҝҳдёҚи¶і50дёӘгҖӮ

hexoеҸҜд»ҘзІ—з•ҘеҲҶдёәдёүдёӘеӯҗйЎ№зӣ®пјҢеҲҶеҲ«жҳҜ:

  • hexo-cli
  • hexo (дёӢж–Үдёӯз”Ёhexo coreжқҘжҢҮд»Ј)
  • hexo plugins

е…¶дёӯhexo pluginsдёҚжҳҜжҢҮжҹҗдёҖдёӘеҚ•зӢ¬зҡ„йЎ№зӣ®пјҢиҖҢжҳҜжіӣжҢҮжүҖжңүзҡ„hexo pluginйЎ№зӣ®гҖӮ

иҜ·зңӢдёӢеӣҫ:

и®©жҲ‘д»¬з»“еҗҲиҝҷеј еӣҫжқҘеӨ§иҮҙзңӢзңӢиҝҷдёүдёӘйЎ№зӣ®зҡ„дҪңз”Ё(дёӢйқўзҡ„й“ҫжҺҘеқҮжҳҜжҢҮеҗ‘Githubдёӯзӣёе…ізҡ„жәҗз Ғ):

  • hexo-cli: hexoе‘Ҫд»ӨиЎҢйЎ№зӣ®пјҢдҪңз”ЁжҳҜ:
    1. еҗҜеҠЁhexoе‘Ҫд»Ө(иҝӣзЁӢ)пјҢеҸҠе…¶еҸӮж•°и§ЈжһҗжңәеҲ¶гҖӮжҜҸж¬ЎжҲ‘д»¬иҫ“е…ҘвҖҷhexo xxxвҖҷе‘Ҫд»ӨеҗҺпјҢйғҪдјҡйҖҡиҝҮnodeи°ғз”Ёhexo-cliдёӯзҡ„entryеҮҪж•°(жҜ”еҰӮпјҢеҸҜд»ҘжҠҠвҖҷhexo initвҖҷи§ҶдёәвҖҷnode hexo-cli/entry.js initвҖҷ)
    2. е®һзҺ°hexoе‘Ҫд»Өзҡ„дёүдёӘеҲқе§ӢеҸӮж•°(еҠҹиғҪ): init / version / help
    3. еҠ иҪҪhexoж ёеҝғжЁЎеқ—пјҢе№¶еҲқе§ӢеҢ–
  • hexo core: hexoж ёеҝғпјҢд»–зҡ„дё»иҰҒдҪңз”ЁеҰӮдёӢ:
    1. е®һзҺ°дәҶhexoеҠҹиғҪжү©еұ•еҜ№иұЎ
    2. е®һзҺ°дәҶhexoж ёеҝғеҠҹиғҪ, еҰӮnew, publish, generateзӯүпјҲе…¶е®һжҳҜдёҖдәӣhexoжҸ’д»¶пјҢдёӢж–ҮдёӯдјҡиҜҰз»ҶеҲҶжһҗпјү
  • hexo plugins: жҢҮдёҖдәӣиғҪеӨҹжү©еұ•hexoзҡ„жҸ’д»¶гҖӮжҸ’д»¶еҸҜд»ҘжҢүеҠҹиғҪеҲҶжҲҗдёӨзұ»:
    1. жү©еұ•hexoе‘Ҫд»Өзҡ„еҸӮж•°пјҢеҰӮhexo-server(е®үиЈ…иҝҷдёӘжҸ’д»¶д»ҘеҗҺжүҚиғҪдҪҝз”Ёhexo serverе‘Ҫд»Ө)
    2. жү©еұ•hexoи§Јжһҗж–Үд»¶зҡ„вҖқиғҪеҠӣвҖқпјҢеҰӮеўһеҠ jadeжЁЎзүҲи§ЈжһҗеҠҹиғҪзҡ„hexo-render-jadeжҸ’д»¶

д»ҺmarkdownеҲ°htmlзҡ„ж—…зЁӢ

з®ҖеҚ•жқҘиҜҙпјҢhexoдёӯпјҢд»ҺmarkdownеҲ°htmlзҡ„generateиҝҮзЁӢдёӯеҒҡдәҶдёӨд»¶дәӢпјҡ

  1. жЁЎжқҝжёІжҹ“
  2. жЁЎжқҝжёІжҹ“

жҳҜзҡ„пјҢе°ұжҳҜиҝҷж ·пјҢе°ұжҳҜдёӨж¬ЎжЁЎжқҝжёІжҹ“гҖӮеҸӘдёҚиҝҮдёӨж¬ЎжёІжҹ“зҡ„иҫ“е…ҘгҖҒжёІжҹ“жЁЎжқҝзҡ„еј•ж“ҺгҖҒиҫ“еҮәдёҚдёҖж ·гҖӮжӯӨеӨ„еә”иҜҘжңүдёҖдёӘиЎЁж јпјҡ

иҝҳеҫ—жңүдёҖеј еӣҫ:

еҜ№дёҠйқўиЎЁж је’Ңеӣҫзҡ„иҜҙжҳҺ:

  • hexo coreеңЁgenerateзҡ„иҝҮзЁӢдёӯдјҡдә§з”ҹдёҖдёӘеҜ№иұЎпјҢжҲ‘д»¬еңЁиҝҷйҮҢжҠҠиҝҷдёӘеҜ№иұЎз§°дёәarticleгҖӮз¬¬дёҖж¬ЎжёІжҹ“зҡ„дё»иҰҒзӣ®зҡ„е°ұжҳҜз»ҷиҝҷдёӘеҜ№иұЎж·»еҠ title,contentзӯүеұһжҖ§гҖӮе…¶дёӯ:
    1. article.title, article.date, article.tags, article.categoriesзӯүеұһжҖ§жқҘиҮӘyml frontзҡ„йғЁеҲҶ
    2. article.contentжҳҜmarkdownж–Үз« и§ЈжһҗеҗҺзҡ„htmlзүҮж®ө
  • hexoйЎ№зӣ®зӣ®еҪ•дёӢеҢ…еҗ«дёүдёӘеӯҗзӣ®еҪ•пјҢ
    1. sourceзӣ®еҪ•пјҢеҶҷеҚҡе®ўзҡ„дё»иҰҒе·ҘдҪңзӣ®еҪ•гҖӮиҝҷдёӘзӣ®еҪ•дёӢеӯҳж”ҫзҡ„жҳҜжҲ‘д»¬зҡ„markdownж–Үз« д»ҘеҸҠjs, images, css
    2. themesзӣ®еҪ•пјҢдё»йўҳзӣ®еҪ•пјҢе®ҡд№үдәҶеҚіе°Ҷз”ҹжҲҗзҡ„htmlзҡ„layout, е’ҢhtmlдёӯйңҖиҰҒеҠ иҪҪзҡ„css, js, images
    3. publicзӣ®еҪ•, hexo generateзҡ„жңҖз»Ҳиҫ“еҮәзӣ®еҪ•гҖӮйҮҢйқўеҢ…еҗ«дәҶж•ҙдёӘеҚҡе®ўзҪ‘з«ҷзҡ„html, css, js, images
  • з¬¬дәҢж¬ЎжёІжҹ“пјҢйңҖиҰҒеј•е…ҘеҜ№еә”жЁЎжқҝж–Үд»¶ж јејҸзҡ„жҸ’д»¶пјҢеҰӮ.ejsж–Үд»¶е°ұйңҖиҰҒдҪҝз”Ёhexo-render-ejsжҸ’д»¶пјҢ.jadeж–Үд»¶йңҖиҰҒдҪҝз”Ёhexo-render-jadeжҸ’д»¶пјҢиҖҢ.sassж–Үд»¶еҲҷйңҖиҰҒhexo-render-sassжҸ’д»¶жқҘиҪ¬жҚўжҲҗcssж–Үд»¶гҖӮhexoзҡ„иҝҷдёҖи®ҫи®ЎжңүзӮ№зұ»дјјwebpackдёӯзҡ„loaderгҖӮ

hexoжҸ’д»¶жҳҜеҰӮдҪ•е·ҘдҪңзҡ„

hexoе’ҢwebpackиҝҳжңүдёҖзӮ№зұ»дјјзҡ„ең°ж–№е°ұжҳҜжҸ’д»¶й©ұеҠЁзҗҶеҝөгҖӮеҚіhexo(е’Ңwebpack)жҳҜе…Ҳе®һзҺ°дёҖеҘ—(жҸ’д»¶)жү©еұ•зі»з»ҹпјҢз„¶еҗҺеҶҚеҫҖжү©еұ•зі»з»ҹдёӯж·»еҠ жҸ’д»¶жқҘе®һзҺ°иҮӘиә«зҡ„еҠҹиғҪгҖӮеҚіжҲ‘д»¬ж—ҘеёёдҪҝз”Ёзҡ„hexo init, hexo newпјҢhexo generateзӯүзӯүеҠҹиғҪйғҪжҳҜйҖҡиҝҮдёҖдёӘдёӘжҸ’д»¶(е…¶е®һе°ұжҳҜдёҖдёӘдёӘfunction)е®һзҺ°зҡ„гҖӮ

е…·дҪ“жқҘи®Іе°ұжҳҜ:

  1. hexo.extendиҝҷдёӘеҜ№иұЎзҡ„жҜҸдёӘеұһжҖ§йғҪжҳҜдёҖдёӘз”ЁжқҘз»‘е®ҡ(зү№е®ҡ)жҸ’д»¶зҡ„еҜ№иұЎгҖӮпјҲжүҖи°“вҖқз»‘е®ҡвҖқпјҢе…¶е®һе°ұжҳҜеҜ№иұЎзҡ„registerж–№жі•пјү
  2. hexoеҲқе§ӢеҢ–иҝҮзЁӢдёӯе…ҲеҠ иҪҪеҶ…йғЁжҸ’д»¶пјҢеҶҚеҠ иҪҪеӨ–йғЁжҸ’д»¶

иҖҢиҝҷдәӣжҸ’д»¶зҡ„еҠҹиғҪеҲҶдёәдёӨеӨ§зұ»: е‘Ҫд»ӨиЎҢжҸ’д»¶е’ҢgenerateиҝҮзЁӢзӣёе…іеҠҹиғҪпјҢдҫӢеҰӮпјҡ

  • е‘Ҫд»ӨиЎҢжҸ’д»¶, hexo new, жҳҜеңЁhexo.extend.consoleеҜ№иұЎдёҠз»‘е®ҡзҡ„дёҖдёӘжҸ’д»¶
  • generateиҝҮзЁӢзӣёе…ізҡ„жҸ’д»¶пјҢеҰӮдёҠж–ҮжҸҗеҲ°зҡ„еҫҖarticleеҜ№иұЎж·»еҠ title,contentзӯүеұһжҖ§зҡ„еҠҹиғҪпјҢжҳҜйҖҡиҝҮеҫҖhexo.extend.processerеҜ№иұЎдёҠз»‘е®ҡpostжҸ’д»¶жқҘе®һзҺ°зҡ„

жүҖд»ҘпјҢеҪ“жҲ‘д»¬жғіиҮӘе·ұеҠЁжүӢеҶҷжҸ’д»¶ж—¶пјҢе°ұжҳҜеғҸhexoе®ҳзҪ‘з»ҷеҮәзҡ„иҝҷж ·,и°ғз”ЁжҹҗдёӘеҜ№иұЎзҡ„registerж–№жі•пјҢеҰӮhexo.extend.console.registerгҖӮ

hexoе’Ңgit page

еҰӮдёҠеӣҫпјҢ(з”ЁжҲ·йҖҡиҝҮжөҸи§ҲеҷЁи®ҝй—®еҲ°зҡ„)git pageдёҠзҡ„еҚҡе®ўзҪ‘з«ҷе…¶е®һжҳҜhexo generateд№ӢеҗҺз”ҹжҲҗзҡ„publicзӣ®еҪ•дёӢзҡ„еҶ…е®№гҖӮ

жүҖд»ҘпјҢдёҖдёӘhexoеҚҡе®ўйЎ№зӣ®еә”иҜҘжңүдёӨдёӘд»“еә“:

  1. (еҹәдәҺhexo initз»“жһңзҡ„)еҚҡе®ўзј–еҶҷд»“еә“гҖӮеҸҜд»ҘжҠҠиҝҷдёӘйЎ№зӣ®зңӢжҲҗдёҖдёӘд»Јз Ғеә“пјҢз”ЁжқҘвҖқејҖеҸ‘вҖқеҚҡе®ўзҪ‘з«ҷ(еҢ…еҗ«еҶҷеҚҡе®ўпјҢз”ҹжҲҗеҚҡе®ўзӯүд»»еҠЎ)
  2. еӯҳж”ҫ(hexo generateз»“жһңзҡ„)publicзӣ®еҪ•д»“еә“гҖӮиҝҷдёӘйЎ№зӣ®жҳҜвҖқеҸӘиҜ»вҖқзҡ„пјҢжҲ‘д»¬дёҚдјҡзӣҙжҺҘдҝ®ж”№иҝҷдёӘд»“еә“зҡ„еҶ…е®№пјҢжҲ‘д»¬д№ҹдёҚдјҡеҜ№иҝҷдёӘд»“еә“зӣҙжҺҘиҝӣиЎҢgit pullгҖҒgit commitгҖҒgit pushзӯүеёёи§„ж“ҚдҪңгҖӮиҝҷдёӘд»“еә“зҡ„еҶ…е®№е°ұжҳҜpublicзӣ®дёӢзҡ„еҶ…е®№пјҢеҚіжҳҜйҖҡиҝҮhexo generateдә§з”ҹгҖҒhexo deployжҸҗдәӨзҡ„гҖӮ

жҖ»з»“

hexoз®ҖжҙҒгҖҒејәеӨ§зҡ„еҠҹиғҪжқҘиҮӘдәҺиҮӘиә«дјҳйӣ…зҡ„зі»з»ҹи®ҫи®Ў:

  1. hexoиҝӣзЁӢеҗҜеҠЁгҖҒhexoж ёеҝғеҜ№иұЎе°ҒиЈ…гҖҒжҸ’д»¶зі»з»ҹеҲҶеҲ«зӢ¬з«Ӣ
  2. иҮӘиә«йҮҮз”ЁжҸ’д»¶й©ұеҠЁпјҢз”ҹжқҘе°ұе…·еӨҮй«ҳеҸҜжү©еұ•жҖ§

еёҢжңӣиҜ»е®ҢиҝҷзҜҮж–Үз« дҪ иғҪеҜ№hexoжң¬иә«жңүжӣҙж·ұе…Ҙзҡ„зҗҶи§ЈпјҢд№ҹиғҪйҖҡиҝҮhexoзҡ„д»Јз Ғи®ҫи®ЎпјҢеҜ№иҮӘе·ұд»ҘеҗҺеҶҷеҮәжӣҙдјҳйӣ…зҡ„д»Јз ҒжңүжүҖеҗҜеҸ‘гҖӮ

еҜ№дәҺжң¬ж–ҮжҲ–иҖ…hexoжңүд»»дҪ•й—®йўҳжҲ–иҜ„и®әпјҢиҜ·еҲ°иҝҷдёӘgit issueйЎөйқўз»ҷжҲ‘з•ҷиЁҖеҗ§гҖӮ
